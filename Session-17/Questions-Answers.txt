Ερώτηση:1. Πως μπορούμε να κάνουμε ένα Request σε ένα εξωτερικό API χωρίς να γνωρίζει ολόκληρο το url μας; 

Απάντηση: Αυτό μπορούμε να το κάνουμε εφόσον έχουμε έναν ενδιάμεσο server με σκοπό να δει το API αυτό το url.

-----------------------------------------------------------------------------------------------------------------
		  
Ερώτηση:2. Αν κάνω ενα GET request και ζητήσω ενα jpeg image, το response τι content-type πρεπει να εχει; 

Απάντηση: Το content-type θα πρέπει να είναι το image/jpeg.

-----------------------------------------------------------------------------------------------------------------

Ερώτηση:3. Πως μπορώ να κάνω ένα GET request και το περιεχόμενο του να κρατηθεί στην cache για 1 μέρα; 

Απάντηση: Σύμφωνα με την πηγή είναι το Cache-Control Header of HTTP με max-age=(Για 1 μέρα σε δευτερόλεπτα αρα 86400 ).

Πηγή :https://theblogreaders.com/restful-web-services-interview-questions-and-answers-part-4/

-----------------------------------------------------------------------------------------------------------------

Ερώτηση:4. Μπορώ να έχω μια εφαρμογή που κάνει μόνο GET requests; 

Απάντηση: Γίνεται εφόσον υπάρχουν ήδη πράγματα για να κάνει GET.

-----------------------------------------------------------------------------------------------------------------
		  
Ερώτηση:5. Μπορώ να έχω μια εφαρμογή που κάνει μόνο POST requests; 

Απάντηση: Ναι γίνεται να έχουμε μόνο POST request σε μια εφαρμογή εφόσον δημιουργούμε περιεχόμενο.

-----------------------------------------------------------------------------------------------------------------
		  
Ερώτηση:6. Τι status code περιμένω από ένα API αν δεν έχω στείλει ένα υποχρεωτικό πεδίο; 

Απάντηση: Εrror 400: Bad Request.

-----------------------------------------------------------------------------------------------------------------
		  
		  
Ερώτηση:7. Τι status code περιμένω από ένα API αν δεν υπάρχει επικοινωνία με τη βάση; 

Απάντηση: Εrror 500: Internal Server Error.

-----------------------------------------------------------------------------------------------------------------

Ερώτηση:8. Μπορώ να κάνω Login με GET request; 

Απάντηση: Δεν γίνεται login με αυτον τον τρόπο.

-----------------------------------------------------------------------------------------------------------------

Ερώτηση:9. Ποια είναι η διαφορά ανάμεσα στο PUT και στο POST; 

Απάντηση: Θα επιλέξουμε την PUT για να κάνουμε update σε αυτό θέλουμε που ήδη υπάρχει στον server από την 
		  επιλογή POST που έχουμε επιλέξει ήδη για να ανεβάσουμε η να δημιουργήσουμε ένα αρχείο.

-----------------------------------------------------------------------------------------------------------------
		 

Ερώτηση:10. Ποια μέθοδο θα πρέπει να χρησιμοποιήσω για να «ανεβάσω» ένα αρχείο;

Απάντηση: Την επιλογή POST.
